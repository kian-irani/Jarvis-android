# ADR-004 — LangGraph for ReAct Agent Orchestration

- **Status:** Accepted
- **Date:** 2026-06-10

## Context
M3 Agentic Alpha needs multi-step agents (plan → tool call → observe → repeat) with persistence, retries, and human-in-the-loop gates.

## Decision
LangGraph state graphs for agent workflows in Brain-Full: each agent is a graph with typed state, tool nodes, and checkpointing to PostgreSQL. ReAct as default pattern; Trust Level System (ADR-007) gates tool nodes.

## Consequences
+ Resumable/inspectable agent runs; checkpoints survive restarts
+ Graph topology makes permissions auditable per node
− Couples agent layer to LangGraph API; wrap behind a port (ADR-002) to keep it swappable

## Alternatives considered
Raw ReAct loop (no persistence); Temporal-only workflows (great for jobs, weak for dynamic LLM branching) — Temporal stays for scheduled/long-running jobs.
